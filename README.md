### :running: Run Locally

Clone the project

```bash

git clone https://github.com/yarinlaniado/Ecommerce-QuickElectric

```

Go to the project directory

```bash

cd my-project

```

Install dependencies

```bash

npm install

```

Start the app

```bash

npm run dev

```

The project run locally on port 3000
